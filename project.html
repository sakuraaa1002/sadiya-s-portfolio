Projets

En un laps de temps et avec la pratique, j'ai pu réfléchir à quel projet réaliser en tant que débutante même si ils ne sont pas encore en cours d'exécution mais je veux bien vous en parler.

Ce site Web est mis en place pour aider les nouveaux apprenants et passionnés de l'informatique à utiliser les outils de la nouvelle technologie à se perfectionner de temps en temps hors de leur cours en classe. On a utilisé comme logiciel VISUAL STUDIO CODE pour créer ce site WEB avec HTML pour la structure des balises comme header pour la tête de page, nav pour la barre de menu en haut de page , main pour le contenu ou corps de la page. Il faut aussi noter qu'on a utilisé des balises de titre comme h1.

Ce site WEB est créé dans le but d'aider les nouveaux bacheliers à choisir leurs filiéres pour les niveaux supérieurs. Ceci permettra à ces derniers de pouvoir faire un bon choix car il y'aura toutes les filières enseignées dans ce pays et ettoutes les universités publiques comme privées où on peut les étudier chacune. Comme toujours on utilisera les mêmes balises que sur le projet précedent et ce sera les mêmes dans le projet qui suit.

Ce site WEB est axé uniquement sur le développement WEB. En effet il permettra de se perfectionneren développement web car en apprenant ça j'ai constaté que les balises étaient un peu difficiles à maîtriser. Donc, il contiendra des tutoriels, des compléments de cours, et même des TP pour plus de pratiques.
